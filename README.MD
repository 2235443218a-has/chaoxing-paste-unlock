## 📖 项目背景
兄弟们，每次写讨论都要手打几百字是不是很崩溃？😡  
学习通网页端的输入框禁止了 `Ctrl+V` 和 `右键菜单`，这个脚本就是为了解决这个反人类设计而生的。

**简单、安全、开源，专为咱们班同学定制。**

## ⚡️ 核心功能
- ✅ **暴力解锁**：强制解除输入框的粘贴限制。
- ✅ **全自动运行**：安装后无需设置，打开学习通自动生效。
- ✅ **安全无毒**：代码完全开源，没有任何数据上传，放心使用。

## ❓ 常见问题 (Q&A)

**Q: 安装后为什么没反应？**
> **A:** 请按 `F5` 刷新一下学习通的页面！脚本只有在页面刷新后才会加载。

**Q: 会被老师发现吗？**
> **A:** 放心，这个脚本只是在你的浏览器本地运行，服务器那边看到的和你手打的一模一样。

### 🚨 紧急避坑指南 (必看！)

**Q: 为什么安装后，右上角的油猴图标还是黑色的/没反应？**
> **A: 这是因为浏览器的安全限制被触发了！请务必检查以下设置：**

1. 在扩展程序里面找到油猴
2. 点击同一栏三个点的图标->点击管理扩展程序
3. 在打开的页面中，找到 **“允许运行用户脚本”** 或类似的开关。(google浏览器)
4. **一定要把这个开关打开！** 开启后刷新网页即可生效。
